# reactive-chat

A real-time chat app built to learn Socket.IO.

## Stack
- **Client**: Next.js 15, Tailwind CSS, shadcn/ui
- **Server**: Express.js, TypeScript, Socket.IO, Mongoose
- **DB**: MongoDB Atlas

## Structure
\`\`\`
reactive-chat/
├── client/     # Next.js frontend
└── server/     # Express + Socket.IO backend
\`\`\`

## Running locally

### Client
\`\`\`bash
cd client && bun dev
\`\`\`

### Server
\`\`\`bash
cd server && bun dev
\`\`\`
